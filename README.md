# Supremo AI

- iOS / iPadOS 26+
- macOS 26+
- visionOS 26+

llama, gemma, phi, gpt2, starcoder, falcon, mpt, bloom, stableLM, qwen, yi, deepseek, mixtral, plamo, mamba, rwkv, gptNeoX, llava, moondream

temperature, greedy, mirostat, mirostatV2, grammar
